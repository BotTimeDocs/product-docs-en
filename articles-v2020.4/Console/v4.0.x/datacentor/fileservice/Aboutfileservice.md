# About File Service

File Service is mainly used to manage all kinds of files, supporting the functions of adding, deleting and modifying folders as well as uploading and downloading files, and providing all kinds of file services to the RPA flow.

## File Service Permission Description

File service permission includes two types:

- **Function Menu Permission**
  
  View File Service: Determine if users can access the File Service page.

- **Data Access Permission**
  
  - Management Permission: Ability to assign user access permissions to folders, including both read-only permission and write permission.
  - Write Permission: Ability to upload and delete files in folders, including read-only permission.
  - Read-only Permission: Ability to view and download files.
